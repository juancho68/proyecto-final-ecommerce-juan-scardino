Las funcionalidades del sitio son:
- Accede a datos externos de un archivo json, donde están los productos.
- Lista dichos productos en la tienda donde es posible seleccionar la cantidad a comprar de cada producto teniendo en cuenta el stock (esto es tenido en cuenta en el scope de la variable que lee el json y no en el archivo)
- Los artículos seleccionados son cargados en el carrito, el cual los graba en el local storage.
- La lógica del carrito admite la actualización de las cantidades en forma dinámica desde el carrito o desde la tarjeta del artículo, realizando el cálculo de la compra.
- Al realizarse la compra de manera exitosa, se actualiza el stock de los productos (dentro de la variable) y de acabarse el stock de algún producto, el mismo se inhabilita.
- El carrito y detalle de cada producto despliega sendos modales para acceder a los datos.
- El formulario de contacto valida el ingreso de datos a través de javascript, teniendo ciertos patrones de datos, ej. validación de mail y teléfono con expresiones regulares.
- Trata de cumplir con diseño responsivo en la mayoría de las páginas y modales.
- Se usó sólo css y javascript, se agregaron fuentes de Google y algunos íconos Font Awesome.
